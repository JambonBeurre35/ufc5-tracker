# UFC 5 Tracker

## Project Overview
This project is designed to track various statistics and gameplay aspects for UFC 5.

## Features
- Track fighter statistics
- Analyze fight outcomes
- Record player performance

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JambonBeurre35/ufc5-tracker.git
   ```
2. Navigate into the project directory:
   ```bash
   cd ufc5-tracker
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start tracking, run:
```bash
yarn start
```
